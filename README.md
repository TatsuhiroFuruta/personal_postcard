# README

## Requirement

* Ruby 3.1.3

* Ruby on Rails 7.2.1

## How to build
 ```
    $ git clone https://github.com/TatsuhiroFuruta/personal_postcard.git
    $ cd personal_postcard
    $ bin/dev
 ```

 Type http://127.0.0.1:3000 in your browser
